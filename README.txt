Coulomb Simulation

Info: Each charge is taken as 1 unit.

Keyboard:
Key		:	Action

Del			Toggle visible lines ON OFF

Return			Toggle Motion ON OFF

End			Charges are Alternate ,ie +,-,+,-,....
			or -,+,-,+,.....

Space			Charges are Random ,ie -,-,-,+,-,+,+

Up			Increase number of charges

Down			Decrease number of charges
